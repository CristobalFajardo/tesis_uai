# Getting started

### Entrenar modelo
`cd model`

`python -m venv env`

`pip install -r requirements.txt`

`python train.py`


### Probar modelo

`cd model`

`python -m venv env`

`pip install -r requirements.txt`

`python test.py`


### Iniciar el servidor

`cd server`

`python -m venv env`

`pip install -r requirements.txt`

`python server.py`


### Iniciar el cliente

`cd frontend`

`npm install`

`npm start`
